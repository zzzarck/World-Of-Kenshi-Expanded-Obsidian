## Project Overview
**World of Kenshi Expanded** is an ambitious, lore-friendly world overhaul mod for Kenshi, designed and developed using the Forgotten Construction Set (FCS). The goal of this project is to push the limits of Kenshi's engine by reworking existing cities, expanding faction interactions, implementing complex multi-stage Town Overrides, and breathing life into barren regions of the map (such as Berserker Country) with lore-accurate points of interest, towns, and camps.

This Obsidian vault serves as the central Game Design Document (GDD), interactive map, and progress tracker for the entire development cycle.

## Key Features of the Mod
- **Dynamic Town Overrides:** Experience a world that reacts to your actions. Cities like [[The Hub]] will transition through multiple stages—from ruined, to rebuilt, to a prospering metropolis—based on world states like defeating local threats or funding reconstruction.
- **Fleshed-out Minor Factions:** Overhauling barebones factions like the [[Faction - Holy Nation Outlaws]] by giving them physical HQs, unique named leaders, custom dialogue trees, and meaningful progression.
- **Zone Expansions:** Filling empty, barren biomes like [[Zone - Berserker Country]] with new, lore-friendly POIs, ruins, and faction strongholds to make exploration rewarding.
- **Lore-Friendly Integration:** Every change, dialogue line, and new location is designed to seamlessly blend into the existing Kenshi universe without breaking established lore.
